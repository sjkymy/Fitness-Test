# 부르스카디오(Bruce-Cardio)
동일 연령대 및 성별 안에서 개인의 심폐체력과 근지구력 수준을 확인할 수 있는 서비스.

+ Vanilla JavaScript로 SPA(Single Page Application) 구현.
+ Chart.js와 D3.js를 이용하여 데이터 시각화.
+ Firebase Cloud를 이용하여 테이터베이스를 만들고 데이터를 저장.
+ 반응형 디자인 구현.
